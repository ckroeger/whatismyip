<h2>Your external IP address is: 201.209.86.139</h2>


<p>
If your computer is on a private network (behind a hardware firewall or router) then your computer's  IP address  is only valid in your own local network.  When you connect to another machine on the Internet, that machine  won't see your internal IP address.  Instead it will see the IP address assigned to your router.  That's what the above IP address is.  </p>

<h2>How do I find my local (internal) IP address?</h2>

<p>The instructions vary based on your computer's operating system.</p>

<h3>For Windows 7 or Vista</h3>

<ul>
<li>Click the <strong>Start Orb</strong><br>
<div style="width: 60px; height:38px; position:relative;" onMouseOver="setStyle('starton', 'zIndex', '20')" onMouseOut="setStyle('starton', 'zIndex', '1')"><img id="startoff" src="win7-start-orb.png" alt="Search" style="position:absolute; top:0; left:0; z-index:10"><img id="starton" src="win7-start-orb-glow.png" alt="Cmd"  style="position:absolute; top:0; left:0; z-index:1"></div></li>
<li>In the <strong>Search</strong> box type <strong>cmd</strong> and press <strong>Enter</strong><br>
<!-- img src="win7-search.png" alt="Search"><br -->
<img src="win7-cmd.png" alt="Cmd"> </li>
<li>In the command window type: <strong>ipconfig</strong> and press <strong>Enter</strong><br>
<img src="ipconfig.gif"></li>
<li>Your IP address will be listed on the <strong>IPv4 Address</strong> line.<br>
<img src="win7-ip-results.png" alt="IP Address Results"></li>
<li>To close the command window type: <strong>exit</strong> and press <strong>Enter</strong><br>
<br>
</li>
</ul>




<h3>For Windows XP or 2000</h3>

<ul>
<li>Click <strong>Start</strong></li>
<li>Click <strong>Run...</strong></li>
<li>In the <strong>Open:</strong> field of the Run window type: <strong>cmd</strong> and click <strong>OK</strong></li>
<li>In the command window type: <strong>ipconfig</strong> and press <strong>Enter</strong></li>
<li>Your IP address will be listed on the <strong>IP Address</strong> line.</li>
<li>To close the command window type: <strong>exit</strong> and press <strong>Enter</strong></li>
</ul>

<h2>I received a message saying 'ipconfig' is not recognized as an internal or external command.  What do I do?</h2>

<p>This message means Windows can't find the ipconfig program. When you type the name of a program at a command prompt, the system looks for that file in the folder you are in. If it doesn't find it, it uses the path. The path is a list of folders that Windows searches to find the program you want to run. By default the Windows and system folders are in the path. If you receive the <b>ipconfig' is not recognized as an internal or external command</b> message, it means the Windows folders aren't in your path.</p>

<p>To work around this, from the command prompt type:</p>

<p><strong>cd\windows\system32</strong><p>

<p>On Windows XP you might need to type this instead:</p>

<p><strong>cd\winnt\system32</strong></p>

<p>Because ipconfig is in the system32 folder you can run it from there.</p>
